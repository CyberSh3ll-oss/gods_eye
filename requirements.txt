# ============================================================
# SECTION 1 — PYTHON RUNTIME
# ============================================================
#
#   Minimum version : Python 3.6+
#   (f-strings are used throughout; 3.6 is the hard floor)
#   Recommended     : Python 3.10+
#
#   Verify your version:
#     python3 --version
#
#   Standard library modules used (ALL built-in, zero pip):
#     os          — screen clearing, file chmod, os.system calls
#     sys         — clean exit handling (sys.exit)
#     subprocess  — spawning terminals, running tools, ping
#     threading   — non-blocking ping in gods_fix()
#     socket      — internet connectivity check
#     time        — DHCP lease wait loop in ip_changer()
#
#   No third-party pip packages are required.
#   Do NOT run: pip install -r requirements.txt
#   This file is documentation only for the Python section.
#
# ============================================================


# ============================================================
# SECTION 2 — CORE RUNTIME SYSTEM DEPENDENCIES
# ============================================================
#
#   These packages are called DIRECTLY by godseye.py at runtime.
#   Without them, core framework functions will fail.
#
#   Install all at once:
#     sudo apt-get install -y python3 git iproute2 iputils-ping \
#       network-manager macchanger xdg-utils xterm bash sudo
#
#   ┌─────────────────────┬──────────────────┬──────────────────────────────────────┐
#   │ Apt Package         │ Binary / Service │ Used For                             │
#   ├─────────────────────┼──────────────────┼──────────────────────────────────────┤
#   │ python3             │ python3          │ Running godseye.py itself            │
#   │ git                 │ git              │ clone_and_run() — Shannon AI clone   │
#   │ iproute2            │ ip               │ ip link set / addr show (ip_changer) │
#   │ iputils-ping        │ ping             │ Connectivity test in gods_fix()      │
#   │ network-manager     │ NetworkManager   │ systemctl restart NetworkManager      │
#   │ macchanger          │ macchanger       │ MAC randomise / restore (-r / -p)    │
#   │ xdg-utils           │ xdg-open         │ Opening URLs in default browser      │
#   │ xterm               │ xterm            │ Fallback terminal emulator           │
#   │ bash                │ bash             │ Shell scripts written to /tmp        │
#   │ sudo                │ sudo             │ Privileged commands throughout       │
#   └─────────────────────┴──────────────────┴──────────────────────────────────────┘
#
# ============================================================


# ============================================================
# SECTION 3 — TERMINAL EMULATORS
# ============================================================
#
#   God's Eye detects the best available terminal in this order:
#     gnome-terminal → xfce4-terminal → konsole → xterm
#
#   At least ONE must be installed. xterm is the guaranteed
#   fallback and should always be present.
#
#   Install preferred + fallback:
#     sudo apt-get install -y xfce4-terminal xterm
#
#   ┌──────────────────────┬─────────────────────────────────────┐
#   │ Apt Package          │ Notes                               │
#   ├──────────────────────┼─────────────────────────────────────┤
#   │ gnome-terminal       │ GNOME desktop default               │
#   │ xfce4-terminal       │ XFCE / Kali default (recommended)   │
#   │ konsole              │ KDE desktop default                 │
#   │ xterm                │ Universal fallback — always install │
#   └──────────────────────┴─────────────────────────────────────┘
#
# ============================================================
